# Smart Loan Management System

A full-stack Smart Loan Management System developed using Angular and Spring Boot.

## Features

- User Registration & Login
- Admin Dashboard
- User Dashboard
- Loan Application System
- Loan Approval Workflow
- Loan History Tracking
- EMI Repayment Module
- Responsive UI Design

## Tech Stack

### Frontend
- Angular
- TypeScript
- Bootstrap

### Backend
- Spring Boot
- Java
- REST API

### Database
- MySQL

## Project Structure

```text
smart-loan-system/
│
├── front-end/
│   └── Angular Application
│
├── back-end/
│   └── Spring Boot Application
```

## How to Run

### Frontend
- cd front-end
- npm install
- ng serve

### Backend
- cd back-end
- mvn spring-boot:run
